# 天气查询工具

一个简洁实用的天气查询工具，提供实时天气信息和未来预报。

## 📝 项目简介

使用 Vue.js 开发的天气查询应用，集成第三方天气 API，提供准确的天气信息和预报。

## ✨ 主要特性

- **🌤️ 实时天气** - 显示当前天气状况
- **📅 天气预报** - 未来 7 天天气预报
- **📍 位置搜索** - 搜索全球城市天气
- **🎨 动态背景** - 根据天气变化背景
- **📱 响应式** - 完美适配各种设备
- **🌍 多语言** - 支持中英文切换

## 🛠️ 技术栈

```
前端框架: Vue 3
开发语言: JavaScript
UI 组件: Element Plus
HTTP 客户端: Axios
天气 API: OpenWeatherMap API
构建工具: Vite
```

## 💻 API 集成

```javascript
// 获取天气数据
async function getWeather(city) {
  const API_KEY = 'your-api-key';
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${API_KEY}&units=metric&lang=zh_cn`;
  
  try {
    const response = await axios.get(url);
    return {
      temperature: response.data.main.temp,
      description: response.data.weather[0].description,
      humidity: response.data.main.humidity,
      windSpeed: response.data.wind.speed,
      icon: response.data.weather[0].icon
    };
  } catch (error) {
    console.error('获取天气失败:', error);
    throw error;
  }
}
```

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/vickiwu/weather-app.git

# 安装依赖
npm install

# 配置 API Key
# 在 .env 文件中添加你的 OpenWeatherMap API Key
VITE_WEATHER_API_KEY=your_api_key_here

# 启动开发服务器
npm run dev
```
