# 个人博客系统

基于 Docusaurus 构建的现代化个人博客和文档网站。

## 📝 项目简介

这是一个使用 Docusaurus 3.x 构建的个人博客系统，支持博客文章、技术文档和项目展示。

## ✨ 主要特性

- **📱 响应式设计** - 完美适配移动端和桌面端
- **🌙 深色模式** - 支持明暗主题切换
- **🔍 全文搜索** - 快速查找内容
- **📊 代码高亮** - 支持多种编程语言
- **🏷️ 标签系统** - 便于内容分类和检索
- **💬 评论系统** - 使用 Giscus 集成 GitHub Discussions

## 🛠️ 技术栈

- **框架**: Docusaurus 3.9.2
- **语言**: TypeScript, React
- **部署**: GitHub Pages
- **CI/CD**: GitHub Actions

## 🚀 核心功能

### 博客系统
- 支持 Markdown 和 MDX 格式
- 自动生成目录和阅读时间
- 标签和分类管理
- RSS 订阅

### 文档系统
- 侧边栏自动生成
- 版本控制
- 多语言支持
- 搜索功能

### 项目展示
- 项目卡片展示
- 详细介绍页面
- 技术栈标签

## 📦 项目结构

```
portfolio/
├── blog/              # 博客文章
├── docs/              # 文档
│   ├── guides/       # 指南
│   └── projects/     # 项目
├── src/              # 源代码
│   ├── components/   # React 组件
│   ├── css/         # 样式文件
│   └── pages/       # 自定义页面
└── static/          # 静态资源
```
