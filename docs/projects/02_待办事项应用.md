# 待办事项应用

一个功能完善的待办事项管理应用，帮助你高效管理日常任务。

## 📝 项目简介

这是一个使用 React 和 TypeScript 构建的待办事项管理应用，支持任务创建、分类、优先级设置和进度跟踪。

## ✨ 主要特性

- **📋 任务管理** - 创建、编辑、删除任务
- **🏷️ 分类标签** - 使用标签组织任务
- **⭐ 优先级** - 设置任务优先级（高/中/低）
- **📅 截止日期** - 设置任务截止时间
- **✅ 进度跟踪** - 实时查看完成进度
- **🔍 搜索过滤** - 快速查找任务
- **💾 本地存储** - 数据持久化保存

## 🛠️ 技术栈

```
前端框架: React 18
开发语言: TypeScript
状态管理: React Hooks (useState, useEffect)
样式方案: CSS Modules
数据存储: LocalStorage
构建工具: Vite
```

## 🎯 核心功能

### 任务创建
- 快速添加任务
- 设置任务详情
- 添加标签和优先级

### 任务管理
- 标记完成/未完成

## 💻 代码示例

```typescript
interface Task {
  id: string;
  title: string;
  description?: string;
  completed: boolean;
  priority: 'high' | 'medium' | 'low';
  tags: string[];
  dueDate?: Date;
  createdAt: Date;
}

const TaskItem: React.FC<{ task: Task }> = ({ task }) => {
  const [completed, setCompleted] = useState(task.completed);
  
  const handleToggle = () => {
    setCompleted(!completed);
    // 更新任务状态
  };
  
  return (
    <div className="task-item">
      <input 
        type="checkbox" 
        checked={completed}
        onChange={handleToggle}
      />
      <span className={completed ? 'completed' : ''}>
        {task.title}
      </span>
    </div>
  );
};
```
